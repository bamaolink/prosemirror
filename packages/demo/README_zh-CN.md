# ProseMirror 演示

该软件包包含一个使用 Vite 构建的 ProseMirror 编辑器的简单演示。它主要用于开发和测试目的。

## 开发

要本地运行此演示，请按照以下步骤操作：

1.  **进入 demo 目录：**
    ```bash
    cd packages/demo
    ```

2.  **安装依赖：**
    ```bash
    npm install
    ```

3.  **运行开发服务器：**
    ```bash
    npm run dev
    ```

这会启动一个 Vite 开发服务器，您可以在浏览器中查看演示。
