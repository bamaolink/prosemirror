# ProseMirror Demo

This package contains a simple demonstration of a ProseMirror editor setup using Vite. It is intended for development and testing purposes.

## Development

To run the demo locally, follow these steps:

1.  **Navigate to the demo directory:**
    ```bash
    cd packages/demo
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

This will start a Vite development server, and you can view the demo in your browser.
